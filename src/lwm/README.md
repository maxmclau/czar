# LWM

Arduino port of [Atmel Lightweight Mesh](http://www.atmel.com/tools/lightweight_mesh.aspx).

### Install

```Shell
$ cd ~/Documents/Arduino/libraries
$ git clone https://github.com/maxmclau/lwm.git
```

### Usage

```Arduino
void setup() {
    Czar.setup();
}
    
void loop() {
    Czar.loop();
}
```
